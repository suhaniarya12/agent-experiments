# AI Agent Showcase

A curated repository of marketing-focused AI agent experiments built on the Ringg platform.

These agents were designed to make AI interactions feel timely, culturally relevant, and immediately useful, with the goal of increasing product curiosity, encouraging trials, and showing how different prompt strategies can shape user engagement.


## Included Showcases

### IPL Agent

A culturally relevant, event-driven agent designed to tap into IPL season energy and turn that attention into more playful, relatable product discovery.

This section is structured to capture:

- the prompt behind the agent
- the user journey from first interaction to conversion intent
- sample conversations
- notes on why the experiment worked

### Mother’s Day Agent

A themed campaign agent built to show how AI can support timely, emotional, and occasion-based engagement in a way that feels contextual rather than generic.

This section is structured to capture:

- the prompt
- landing page copy
- the intended use case

### Personal Assistant Agent

A utility-first agent designed to feel broadly relatable and practical, making it easier for users to imagine everyday use cases for conversational AI.

This section is structured to capture:

- the prompt
- the workflow
- the operational logic of the experience

## Purpose

This repo is meant to serve as a lightweight showcase of agent thinking rather than just a prompt archive.

Each experiment is documented not only by what was built, but also:

- why that framing was chosen
- what kind of user behavior it was designed to unlock
- how the flow was structured
- what made the interaction resonate

## How To Use This Repo

Use this repo to:

- document campaign-style AI agent experiments
- present prompt strategy in a structured format
- share standout examples with teammates, prospects, or partners
- build a reusable library of high-performing AI interaction patterns
